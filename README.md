# Дипломный проект профессии «1C-программист: с нуля до middle» (Коробка А.А.)
Реализовано в полном объеме
Кастомизация конфигурации «Управление IT-фирмой для компании «Ваш компьютерный мастер»

# Цель дипломного проекта
Разработать и подготовить к эксплуатации новый функциональный блок в конфигурации «Управление IT-фирмой»:
реализовать новые подсистемы «Обслуживание клиентов» и «Расчёт зарплаты» в соответствии с требованиями заказчика,
подготовить отчёты по новой функциональности,
оформить инструкцию по начальной настройке,
подготовить автотест для проверки корректности работы подсистемы «Обслуживание клиентов»,
загрузить результат работы в новый репозиторий на GitHub.

# Решение
## Описание нового функционала
## 1. Уведомления в Telegram
Создать группу и бота в telegram. Сохранить в константу "Токен управления ТГ ботом" - токен бота, "Идентификатор группы для оповещения" - идентификатор группы. В группу будут приходить уведомления о вновь созданных документах "Обслуживание клента" и при изменении в существующих даты проведения работ или специалиста.

## 2. Заполнение и печать документа "Реализация товаров и услуг"
Для договоров с видом "Абонентская плата" добавлена возможность заполнения ежемесячной абонентской платы и суммы за выполненные работы в табличной части "Услуги" документа "Реализация товаров и услуг". Для этого необходимо установить в константах "Номенклатура абонентская плата" и "Номенклатура работы специалиста" соответствующие значения справочника "Номенклатура".

## 3. Расчет зарплаты
Для работы с зарплатными документами предварительно нужно установить оклад и процент оплаты выполненных работ для каждого сотрудника в регистре сведений "Условия оплаты труда", заполнить график работы с помощь обработки "Заполнение графика работы", указав период заполнение графика и выходные дни.

## 4. Создать профили групп доступа со следующими ролями:
Бухгалтер ИТ фирмы - Базовые права БСП, Вывод на принтер, в файл, в буфер обмена, Массовое создание актов, и т.д. полный список прав можно посмотреть в Админстрирование-Настройка пользователей и прав-Профили групп доступа;
Кадровик-расчетчик - Базовые права БСП,  Вывод на принтер, в файл, в буфер обмена, Работа с кадровыми документами, и т.д. полный список прав можно посмотреть в Админстрирование-Настройка пользователей и прав-Профили групп доступа;
Менеджер - Базовые права БСП, Добавление документов обслуживание клиента, и т.д. полный список прав можно посмотреть в Админстрирование-Настройка пользователей и прав-Профили групп доступа;
Специалист - Базовые права БСП, Проведение документов обслуживание клиента, Выработка специалистов, и т.д. полный список прав можно посмотреть в Админстрирование-Настройка пользователей и прав-Профили групп доступа;
